AGGIORNAMENTO ALLA VERSIONE 2.0.0

1. Estrai questo ZIP.
2. Copia tutti i file estratti dentro:
   C:\Progetti\zima-storage-manager
3. Conferma la sostituzione dei file.
4. Apri PowerShell nella cartella del progetto.
5. Esegui:

git add .
git commit -m "Release Zima Storage Manager 2.0"
git push

Dopo il push, installa o aggiorna ZSM sul server.

Nuova installazione:
curl -fsSL https://raw.githubusercontent.com/AngoloInformatico/zima-storage-manager/main/install.sh | sudo bash

Aggiornamento di un'installazione v2:
sudo zsm-update
