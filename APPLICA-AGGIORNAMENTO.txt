ZIMA STORAGE MANAGER 3.0.0-RC1

1. Estrai l'archivio.
2. Copia il contenuto della cartella zima-storage-manager nel repository locale.
3. Controlla le modifiche:

   git status
   git diff --stat

4. Pubblica sul ramo develop-v3:

   git checkout develop-v3
   git add .
   git commit -m "Release candidate Zima Storage Manager v3.0.0-rc5"
   git push origin develop-v3

TEST CONSIGLIATI PRIMA DELLA RINOMINA

   sudo zsm status
   sudo zsm disks
   sudo zsm audit
   sudo zsm backup

INSTALLAZIONE DEL RAMO DI TEST

   curl -fsSL https://raw.githubusercontent.com/AngoloInformatico/zima-storage-manager/develop-v3/install.sh | sudo ZSM_BRANCH=develop-v3 bash

Aprire quindi http://IP_ZIMAOS:8765 e verificare la pagina Diagnostica.
