ZIMA STORAGE MANAGER 2.1.0 - RELEASE CANDIDATE

1. Conserva una copia della cartella attuale del progetto.
2. Copia il contenuto di questo pacchetto nella cartella locale del repository.
3. Controlla le modifiche:

   git status
   git diff

4. Pubblica:

   git add .
   git commit -m "Release Zima Storage Manager v2.1.0"
   git push

5. Sul server ZimaOS esegui l'installazione nativa consigliata:

   curl -fsSL https://raw.githubusercontent.com/AngoloInformatico/zima-storage-manager/main/install.sh | sudo bash

6. Prima della prima rinomina verifica:

   sudo zsm status
   sudo zsm disks
   sudo zsm audit

È incluso anche un pacchetto Docker sperimentale. Leggi docs/DOCKER.md prima di usarlo.
